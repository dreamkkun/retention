# 보안 설정 가이드

## 🔒 보안 기능

### 1. 사용자 인증
- 이름, 부서, 사번 입력 필수
- 접속 정보 로컬 스토리지 저장
- 모든 접속 기록 로깅

### 2. IP 화이트리스트
`backend/app.py`의 `ALLOWED_IPS` 수정:

```python
ALLOWED_IPS = [
    '127.0.0.1',
    'http://localhost:3000/',
    '192.168.1.',    # 사내 IP 대역
    '10.0.0.',       # 사내 IP 대역
    '172.16.0.',     # 사내 IP 대역
]
```

### 3. 세션 타임아웃
- 기본: 30분 후 자동 로그아웃
- 5분 전 경고 메시지
- 사용자 활동 시 자동 연장

### 4. 워터마크
- 화면 전체 "대외비" 워터마크
- 사용자 정보 (이름, 부서, 사번) 표시
- 접속 시간 표시

### 5. 복사/스크린샷 방지
- 텍스트 선택 불가 (`user-select: none`)
- 우클릭 방지
- 드래그 방지
- 인쇄 시 워터마크 추가

### 6. 접속 로그
- 모든 접속/업로드 기록
- IP, 시간, 사용자 정보 저장
- `backend/access_logs.json`에 저장
- 최근 1000개 기록 유지

## 🛡️ 추가 보안 설정 (선택사항)

### 1. HTTPS 강제
프로덕션 환경에서는 반드시 HTTPS 사용

### 2. VPN 필수
사내망 접속 시 VPN 연결 필수화

### 3. 2단계 인증
SMS/OTP 인증 추가

### 4. 데이터베이스 연동
- 사용자 정보 DB 검증
- 접속 권한 관리
- 실시간 접속 모니터링

## 📋 접속 로그 확인

```bash
# 백엔드 서버에서
cat backend/access_logs.json
```

또는 API 호출:
```bash
curl http://localhost:5000/api/access-logs
```

## ⚠️ 주의사항

1. **IP 화이트리스트 설정 필수**
   - 프로덕션 환경에서는 반드시 사내 IP만 허용

2. **접속 로그 정기 점검**
   - 비정상 접속 패턴 모니터링
   - 주기적으로 로그 백업

3. **세션 타임아웃 준수**
   - 자리 비울 때 반드시 로그아웃
   - 브라우저 종료 시 자동 로그아웃

4. **화면 캡처 금지**
   - 스크린샷, 화면 녹화 금지
   - 위반 시 징계 대상
