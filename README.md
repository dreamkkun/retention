# Retention Policy Support System

리텐션 정책 지원 시스템 - 고객 상담사를 위한 인터넷/TV 정책 조회 및 혜택 계산

## 🚀 배포된 사이트

배포 URL: [Vercel에서 확인 예정]

## 📖 사용 방법

### 일반 사용자
1. **전체 정책 보드**: 번들, 디지털, 동등결합, D단독 정책 조회
2. **맞춤형 혜택 계산기**: 고객 요금 입력 후 실시간 혜택 계산

### 관리자
1. **관리자 탭** 클릭
2. **로그인**: `admin` / `retention2026`
3. **템플릿 다운로드**: 5개 시트 엑셀 파일
4. **정책 업데이트**: 엑셀 파일 업로드 → JSON 생성

## 🛠️ 로컬 개발

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start

# 프로덕션 빌드
npm run build
```
