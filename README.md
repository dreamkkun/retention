# 리텐션 정책 지원 시스템

고객 상담사를 위한 인터넷/TV 리텐션 정책 조회 및 혜택 계산 시스템입니다.

## 주요 기능

### 1. 전체 정책 보드
- 엑셀 스타일의 심플한 정책 테이블
- 번들 재약정, 디지털 재약정, 동등결합, D단독 정책 조회
- 추천 구간 시각화 (파란색 박스)
- 카테고리별 필터링

### 2. 맞춤형 혜택 계산기
- 고객 유형 선택 (번들/D단독)
- 현재 요금 입력 → 자동 판가 구간 판정
- 요금제 변경 옵션 선택
- 실시간 혜택 계산 (상품권, 할인)
- 방어 멘트 자동 생성

### 3. 관리자 대시보드
- 관리자 로그인 (권한 관리)
- 엑셀 파일로 정책 업로드
- 이미지 파일 업로드
- 정책 템플릿 다운로드

## 시작하기

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm start
```

브라우저에서 `http://localhost:3000` 접속

### 빌드
```bash
npm run build
```

## 관리자 기능

### 로그인 정보
- 아이디: `admin`
- 비밀번호: `retention2026`

### 정책 업데이트 방법

#### 1. 엑셀 파일 업로드
1. 관리자 대시보드에서 "템플릿 다운로드" 클릭
2. 엑셀 파일에서 정책 데이터 수정
3. "정책 업로드" 섹션에서 수정된 파일 업로드
4. 생성된 JSON 파일 다운로드
5. `src/data/policies.json` 파일을 다운로드한 JSON으로 교체
6. 페이지 새로고침

#### 2. 이미지 파일 업로드
1. 정책 이미지 파일 업로드
2. `public/assets/` 폴더에 이미지 복사
3. `src/data/policies.json`의 `policy_images` 섹션 업데이트

## 파일 구조

```
retention/
├── public/
│   └── assets/              # 정책 이미지 파일
├── src/
│   ├── components/
│   │   ├── PolicyBoard.js        # 정책 보드 (엑셀 스타일)
│   │   ├── BenefitCalculator.js  # 혜택 계산기
│   │   ├── AdminLogin.js         # 관리자 로그인
│   │   └── AdminDashboard.js     # 관리자 대시보드
│   ├── data/
│   │   └── policies.json         # 정책 데이터
│   ├── App.js
│   └── index.css
└── package.json
```

## 기술 스택

- **React 18**: UI 라이브러리
- **Tailwind CSS**: 스타일링 (심플한 회색조 디자인)
- **xlsx**: 엑셀 파일 처리

## 디자인 컨셉

### 심플한 엑셀 스타일
- 회색조 배경 (`#F5F5F5`)
- 흰색 테이블 배경
- 회색 테두리 (`#D1D5DB`)
- 추천 구간만 파란색 박스로 강조

- **테두리**: `#D1D5DB` (회색)
- **헤더**: `#F3F4F6` (연한 회색)
- **강조**: `#2563EB` (파란색 - 추천 구간)
- **텍스트**: `#1F2937` (진한 회색)
